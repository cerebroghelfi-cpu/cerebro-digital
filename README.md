# Cerebro Digital

Tu segundo cerebro: notas, voz, dibujos, fotos, enlaces — conectados como un grafo.

## Para usarlo en tu móvil (PWA gratis)

Sigue las instrucciones del chat con Claude paso a paso. Resumen:

1. Sube esta carpeta a un repositorio de GitHub
2. Conéctala a Vercel
3. Vercel te da una URL pública
4. Abres esa URL en Chrome del móvil
5. Menú → "Instalar aplicación" / "Añadir a pantalla de inicio"

¡Listo! Ya tienes app con icono propio que funciona offline.

## Para desarrollar local (opcional)

```
npm install
npm run dev
```
